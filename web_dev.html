Ingredients to a good web cake

Websites come in all shapes and sizes. Everything from the most simple single page websites, to advanced Angular bootstrap applications. All of these websites have something in common. They require the programmer to know Javascript, CSS, and HTML. Sometimes, in the case of Angular, the programmer also needs to know TypeScript, a superset of JavaScript. For more advanced websites, like ones that update in real time and require client to client connections, frameworks and programming languages like jQuery, Angular, socketio, and MySQL are nessessary. I have personal experience with JavaScript, CSS, HTML, Angular, and Socket.io, but still have yet to try MYSQL and jQuery. MYSQL serves as the storage system for a website, managing and storing all the data that goes through. jQuery is a JavaScript library that makes programming easier by abstracting concepts and methods.
